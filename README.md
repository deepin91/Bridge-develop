## 기술 스택  

**Backend**: Java, JavaScript, Spring Boot(3.2.5), Spring Data JPA, MyBatis, Redis, JWT  
**Frontend**: React, HTML, CSS  
**Database**: MySQL, H2 (테스트)    
**API 문서화**: springdoc-openapi (Swagger UI)  
**Etc**: Gradle, GitHub
